# Практическая работа №7 часть 2

Выполнили: студентки группы 3 ИСиП-423 Кузьмина Анна и Цветнова Арина

Вариант 1: Шифр Цезаря

Метод: классический шифр сдвига символов на фиксированное число
позиций.

Задача: реализовать модуль с методами EncryptCaesar(string text, int
shift) и DecryptCaesar(string cipher, int shift).

Требования:
- поддержка русского и английского алфавитов;
- сохранение регистра и неалфавитных символов.

Тестирование:
- позитивные тесты (разные сдвиги, языки);
- краевые случаи (сдвиг = 0, сдвиг &gt; длины алфавита);
- обратимость шифрования (encrypt → decrypt).
